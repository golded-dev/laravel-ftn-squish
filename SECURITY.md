# Security

This package reads old message-base files. Most bugs will be boring parsing bugs, but reports still need a clean path.

## Supported Versions

| Version | Supported |
| --- | --- |
| `1.x` | Yes |

## Reporting A Vulnerability

Do not open a public issue for a security problem.

Report it privately through GitHub Security Advisories for this repository:

https://github.com/golded-dev/laravel-ftn-squish/security/advisories/new

Include:

- what is affected
- how to reproduce it
- what impact you expect
- whether you have a suggested fix

Expect a human response before drama. If the report is valid, the fix will ship with a patched release and a changelog entry.
